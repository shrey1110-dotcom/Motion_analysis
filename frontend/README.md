# Frontend

React + Vite client for live camera analysis, upload analysis, and authenticated session history.

## Stack
- React 18
- Vite
- Clerk
- Chart.js
- Three.js

## Local Development
```bash
cp .env.example .env
npm install
npm run dev
```

Set `VITE_CLERK_PUBLISHABLE_KEY` in `.env` before signing in locally.
